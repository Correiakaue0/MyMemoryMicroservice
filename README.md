# Harmonit MyMemory Integration

Microserviço responsável pela integração com a API do MyMemory, permitindo tradução automática de textos entre diferentes idiomas.

## Funcionalidades

- Tradução de textos utilizando MyMemory
- Suporte a múltiplos idiomas
- API REST para integração entre sistemas
- Estrutura preparada para microsserviços
- Fácil configuração via environment variables

---

## Tecnologias Utilizadas

- .NET
- ASP.NET Core
- REST API
- MyMemory Translation API
